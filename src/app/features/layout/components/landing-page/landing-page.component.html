Explora, Mide, Evoluciona.

GeoUAX es tu asistente ambiental personal. Desde estudiantes hasta exploradores urbanos, nuestra herramienta te permite optimizar cada trayecto y compartir el cambio.

Puntos sostenibles

Guarda ubicaciones, imágenes y datos ambientales. Todo se registra en la nube con tecnología Firebase.

Rutas por tipo de transporte

Elige entre a pie, bicicleta o coche. Visualiza el impacto y selecciona la ruta más responsable.

Huella de carbono

GeoUAX calcula las emisiones de cada trayecto y te ofrece alternativas para reducirlas.

Retos y logros

Completa retos semanales, sube de nivel y colabora con la comunidad para mejorar el mundo.
